# Codolm
Presentation website where I offer face-to-face teaching hours in the programming languages I know.

[Check Out](https://codolm.vercel.app/)

## Procedure
    npm create vite@latest
    cd codolm
    npm install
    npm run dev
    npm install styled-components@5.3.10
